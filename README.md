# An-LLM-Based-Resume-Intelligence-System-For-Recruiters-and-Job-Seekers
AI-powered Resume Intelligence System that uses LLMs, semantic embeddings, and FAISS to analyze resumes, match them with job descriptions, rank candidates, and provide ATS scoring with skill gap insights and feedback for job seekers.

This project is an LLM-Based Resume Intelligence System designed to automate resume analysis for both recruiters and job seekers. The system leverages Large Language Models (FLAN-T5), NLP techniques, semantic embeddings, and FAISS-based vector search to evaluate resumes against job descriptions.

It supports two modes:

Recruiter Mode – ranks candidates based on job-role similarity
Job Seeker Mode – provides ATS score, skill gap analysis, and personalized feedback

The system follows an end-to-end pipeline including text extraction, resume parsing, embedding generation, similarity computation, and result visualization through an interactive Gradio interface.
